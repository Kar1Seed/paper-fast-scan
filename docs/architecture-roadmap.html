二、功能路线图 Roadmap

版本功能类别优说明
v2.0 ✅ PDF 论文解析 (pymupdf)📄 解析P0 文本提取 · 图表分离 · LaTeX 识别 · 论文元数据
双模板 (page.html + paper.html)📤 输出P0 Web 白皮书用 page.html，学术论文用 paper.html
v2.1 🚧 Glossary 交叉引用 & hover 双链📤 输出P0 正文术语 hover 浮窗显示释义；侧栏 Glossary 点击跳转正文
多 Agent 协作架构设计🧠 理解P1 主 Agent 调度 + 检索/图表/校对子 Agent 并行
安装脚本 (install.py)🔧 工具P0 一键安装依赖，新人友好
示例库扩充 (≥5 篇)📚 文档P1 覆盖 DEX / L1 / DeFi / AI / 零知识证明 多领域
v2.2 Glossary 知识库持久化💾 数据P0 SQLite 存储术语 · 释义缓存 · 跨论文复用 · 避免重复解释
交互式 HTML: 折叠章节📤 输出P1 长页面支持 expand/collapse，默认展开核心章节
暗/亮色主题切换📤 输出P2 尊重用户阅读偏好 · 记忆选择
v2.3 作者数据库💾 数据P1 arXiv API 自动抓取作者信息 · 机构 · 论文列表
Notion / Obsidian 文档源📥 输入P2 通过 API 接入 Notion 页面和 Obsidian vault
格式注册中心 (插件式)🔧 解析P1 新文档格式通过注册即可支持，不修改核心代码
作者合作网络图📤 输出P2 SVG 展示同一作者跨论文关系 · 机构聚类
v2.5 多 Agent 编排正式上线🧠 理解P1 Hermes delegate_task 并行 3 个 Agent: 翻译/图表/校对
导出格式扩展 (EPUB/JSON)📤 输出P2 PDF 打印版 · EPUB 电子书 · JSON 结构化数据
结构化提取 (表格·公式·引文)🔧 解析P1 PDF 表格→HTML table · 引文→可点击链接
v3.0 论文关联知识图谱💾 数据P1 跨论文引用链 · 主题聚类 · 时间线可视化
视频/播客转文字📥 输入P2 Whisper 语音转文字 → 结构化摘要
社区模板市场📤 输出P2 用户贡献自定义模板 · 投票排名
